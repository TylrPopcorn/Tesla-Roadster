# ```Tesla-Roadster```
```A simple hero page created with Bulma CSS```

This project demonstrates the basics of the Bulma CSS framework through the creation of a simple hero page. Designed to showcase the power of Bulma, the page highlights Tesla's Roadster, combining sleek design with responsive elements. Perfect for anyone looking to explore the potential of Bulma's layout and syntax system.

Live: https://tesla-roadster.onrender.com/
